URL Pública do Site

A API da Meta acessa as fotos diretamente pela URL pública do seu servidor. O site precisa estar hospedado com HTTPS. Não funciona em localhost.

Sem barra no final. Ex: https://meusite.com.br

Instagram Business

{% if config.ig_configured %} Configurado{% endif %}
  1. Vá em developers.facebook.com → Seu App → Configurações Básicas
  2. Em "Produtos", adicione Instagram Graph API
  3. Conecte uma conta Instagram Business/Creator à sua Página do Facebook
  4. Gere um User Access Token com escopo instagram_content_publish,instagram_basic
  5. No Instagram Graph API, chame GET /me/accounts para obter o Instagram User ID

Facebook Page

{% if config.fb_configured %} Configurado{% endif %}
  1. No painel do desenvolvedor, adicione o produto Facebook Login ao seu App
  2. Solicite a permissão pages_manage_posts e pages_read_engagement
  3. No Graph API Explorer, gere um token com estas permissões e troque-o por um Page Access Token
  4. O Page ID aparece em "Sobre" na sua Página do Facebook, ou via GET /me/accounts
